# ¡Bienvenidos a mi primer post!

¡Hola! Este es un hito importante para mí. Hoy estreno oficialmente este espacio digital maquetado y construido con **Astro**, el framework moderno que está cambiando la forma en que creamos sitios web.

## ¿Por qué elegí Astro?

Decidir qué tecnología usar para un blog no siempre es fácil. Elegí Astro por tres razones principales:

* **Rendimiento brutal:** Astro elimina el JavaScript innecesario por defecto.
* **Arquitectura de Islas:** Puedo usar componentes de React o Vue solo donde los necesito.
* **Experiencia de desarrollo:** Escribir en Markdown es nativo, rápido y cómodo.
